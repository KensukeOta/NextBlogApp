# NextBlogApp
Next.jsのApp Routerを使用した完全SPAのブログアプリです。

## 使用した技術・ツール
- 開発環境: __Docker__, __docker compose__
- バックエンド: __Laravel10.27.0__

  https://github.com/KensukeOta/API-for-NextBlogApp
- フロントエンド: __Next.js13.5.4__
- CSSフレームワーク: __Tailwind CSS3.3.3__
- フォームライブラリ: __React Hook Form7.47.0__
- マークダウンライブラリ: __react-markdown9.0.0__, __remark-gfm4.0.0__, __github-markdown-css5.3.0__
- アイコンライブラリ: __Bootstrap Icons1.11.1__
- UIコンポーネントエクスプローラー: __Storybook7.4.6__
- 認証ライブラリ: __Laravel Sanctum3.3.1__
- その他ライブラリ: __js-cookie3.0.5__
- コンポーネント設計: __Atomic Design__
- バージョン管理ツール: __Git__ / __GitHub__
- タスク管理: __GitHub Issues__
- その他言語: __React18.2.0__, __TypeScript5.2.2__, __HTML/CSS__

## 機能
- ログイン機能
- ユーザー登録機能
- 記事投稿機能
- 記事一覧表示機能
- 記事詳細表示機能
- 記事更新機能
- 記事削除機能
